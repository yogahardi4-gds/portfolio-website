## Portfolio Website
A modern, responsive personal portfolio website built with HTML, CSS, JavaScript, and Bootstrap 5.3.2.
This website showcases projects, skills, and contact information in a clean and user-friendly design, perfect for developers, designers, or anyone who wants to present their work online.

## Features
- Fully responsive layout using Bootstrap 5.3.2
- Sections for About, Projects, Skills, and Contact
- Smooth scrolling and interactive elements with JavaScript
- Easy to customize and extend
- Clean and simple design suitable for professional portfolio
